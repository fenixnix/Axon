> **版本**: 2.0.0

## 1. 验收总览

本文档定义 Axon 项目每个阶段的验收标准。每个任务完成后需要通过对应的验收检查。

**验收原则**:
- ✅ 功能完整 - 所有预期功能已实现
- ✅ 代码质量 - 通过 lint 和 format 检查
- ✅ 测试覆盖 - 核心功能有测试保护
- ✅ 文档同步 - 关键变更已更新文档

## 7. 验收流程

### 7.1 开发自检清单

每次提交前执行：

```bash
# 1. 格式检查
cargo fmt

# 2. Lint 检查
cargo clippy -- -D warnings

# 3. 测试
cargo test

# 4. 文档生成 (可选)
cargo doc --no-deps
```

### 7.2 发布检查清单

每个 Phase 完成时执行：

```bash
# 完整检查
cargo fmt
cargo clippy -- -D warnings
cargo test
cargo build --release

# 安全审计 (可选)
cargo audit

# 检查二进制
ls -lh target/release/axon
```

## 8. 缺陷管理

### 8.1 严重缺陷 (Blocking)

- 编译失败
- 核心功能无法使用
- 数据丢失风险
- 安全漏洞

### 8.2 主要缺陷 (Major)

- 非核心功能异常
- 性能严重下降
- 用户体验问题

### 8.3 次要缺陷 (Minor)

- 文档错误
- 边缘 case
- 改进建议
